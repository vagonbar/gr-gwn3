gr-gwn3.python package documentation

Part of gr-gwn3

GNU Radio GWN3 Python blocks and tests.

For the GWN3 library, please see the libgwn directory. Pure Python blocks require an import in this file __init__.py, of the form:

    from .block_name import block_name

This is necessary to have the new block included in the gwn3 package.

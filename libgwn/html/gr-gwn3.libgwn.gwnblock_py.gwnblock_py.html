The GWN basic block constructor.

The GWN basic block implements facilites used by all GWN blocks. A descendent of this class may call this constructor to fix the number of ports, timers and timeouts.

Parametersnamea name for this block.
number_innumber of input ports. Input ports are called in_0, in_1, etc.
number_outnumber of output ports. Output ports are called out_0, out_1, etc.
number_timersnumber of internal timers.
number_timeoutsnumber of internal timeouts.
